# Redis的数据结构

>  Redis 是一个key-value的数据库，key一般是String类型，value的类型多种多样。

| 类型          | 示例                    |
| ------------- | ----------------------- |
| **String**    | hello world             |
| **Hash**      | {name: "Jack", age: 21} |
| **List**      | [A -> B -> C -> C]      |
| **Set**       | {A, B, C}               |
| **SortedSet** | {A:1, B:2, C:3}         |
| **GEO**       | {A:(120.3, 30.5)}       |
| **BitMap**    | 0110110101110101001     |
| **HyperLog**  | 0110110101110101011     |
